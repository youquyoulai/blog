# 2026-04-27

## 博客 URL 和评论系统调试

### 最终状态
- **hugo.toml**: 添加了 `[permalinks] posts = '/archives/:slug.html'`，统一所有文章 URL 格式
- **Waline 评论 path 逻辑**: `pathname.endsWith('.html') ? pathname : pathname + '.html'`，兼容带/不带 .html 的情况
- **Cloudflare Pages**: 会自动 308 重定向去掉 .html 后缀，无法关闭（`wrangler.toml` 和 `_headers` 的 `html_handling = "none"` 均无效）
- **文章 url 字段**: 保留了所有文章的 `url: /archives/xxx.html` 手动字段（删除会导致无 slug 的文章生成中文 URL）

### 踩坑记录
- 批量删除文章 `url` 字段会导致无独立 `slug` 字段的文章用中文标题生成 URL，不可行
- Cloudflare Pages 的 308 去掉 .html 行为是平台内置的，`wrangler.toml` 的 `[assets] html_handling = "none"` 只对 Workers 项目有效
- `_headers` 文件的 `X-HTML-Handling: none` 对 Pages 项目无效
- Cloudflare 重定向规则会造成无限循环，需避免
- `public/archives/xxx.html` 有时会变成目录而非文件，需要彻底删除 public 目录再重新构建
