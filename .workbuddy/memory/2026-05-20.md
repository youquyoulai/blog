## ⚠️ 踩坑：Hugo 自定义输出格式导致文章变成 XML
- 错误做法：在 `[outputs]` 中把 `sitemap` 加入 `page/home/section`，并在 `layouts/sitemap.xml` 放自定义模板
- 后果：Hugo 把 sitemap 模板当成 page 的 HTML 模板，所有文章渲染为 XML
- **正确做法**：只用 `[sitemap]` 配置块控制，Hugo 内置会自动生成 `sitemap.xml`，不需要任何 `[outputs]` 修改，也不需要自定义模板
