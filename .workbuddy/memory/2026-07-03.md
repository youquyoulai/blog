# 2026-07-03 工作日志

## 知识库改造：博客首页 → 分类卡片 + 三级层次结构

将 Hugo 博客从传统博客列表首页改造为知识库风格，支持「分类 → 人物 → 作品」三级层次。

### 新增文件
- `data/kb-categories.yaml` — 5 个分类元数据（数学📐、文学📚、历史🏛️、哲学🧠、科学🔬）
- `layouts/index.html` — 知识库首页（分类卡片网格 + 最近更新列表）
- `layouts/_default/section.html` — 智能 section 模板（有子 section 显示人物卡片，无子 section 显示文章列表）
- `layouts/_default/author.html` — 作者详情页模板（照片 + 生平 + 成就 + 作品列表）
- `layouts/partials/header.html` — 覆盖主题 header，导航栏改为 KB 分类链接
- 示例内容：`content/literature/`（王小波、闫连科两位作者 + 2 篇作品评价）
- 空分类 section：`content/history/`、`content/philosophy/`、`content/science/`

### 修改文件
- `hugo.toml`：mainSections 增加 5 个分类、permalinks 增加 `:sections` 规则、description 更新
- `content/math/_index.md`：title 从"数学题解"改为"数学"

### 关键技术点
- Hugo 模板覆盖：根目录 `layouts/` 优先于 `themes/weisaygrace/layouts/`
- `layout: author` frontmatter 触发 `layouts/_default/author.html` 模板
- `.Sections` 判断 section 是否有子 section（人物），`.Pages` 获取文章列表
- `:sections` permalink 变量生成嵌套 URL（如 `/literature/wang-xiaobo/slug.html`）
- `(index .Site.Data "kb-categories")` 访问带连字符的数据文件名
- `print . " 年生"` 替代 `printf "%s"` 处理 YAML 数值类型
- CSS 内联在模板中，复用主题 CSS 变量（`--color_link`、`--color_border` 等），自动适配暗色模式

### admin 后台兼容性修复（关键！）
原有 admin 不兼容知识库改造后的子目录结构，主要问题：
1. **listPosts 只扫描扁平目录** — 数学文章移入子目录后，admin 列出 0 篇
2. **getPost/updatePost/deletePost 路径构造不含子目录** — 编辑/删除失败
3. **分类下拉只有 math/posts** — 缺少其他分类
4. **getTaxonomies 只扫描扁平目录 + 域名硬编码 math.pgoj.top**

修复方案：
- **worker.js**: listPosts 递归扫描子目录，返回 `subSection` 字段；getPost/updatePost/deletePost/createPost 支持 `subsection` 查询参数；getTaxonomies 递归扫描 + 域名改 www.pgoj.top
- **index.html**: 分类下拉增加全部 6 个分类；editPost/savePost/deletePost 传递 `subsection` 参数；分类/标签中文映射更新
- ⚠️ **worker.js 需重新部署到 Cloudflare Workers** 才能生效（通过 Dashboard 或 wrangler）
