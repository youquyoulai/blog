## 一、题目呈现

**题目内容：**

已知数列 $\{a_n\}$ 满足 $a_{3n} = n^2 + n$（$n \in \mathbb{N}^*$）。若 $\{a_n\}$ 中存在连续9项构成等比数列，则该等比数列公比 $q$ 的最大值为 ______。

**答案：$\sqrt{2}$**

---

## 二、命题意图分析

### 1. 从解答大题移至填空压轴，考查方式革新

在过去的几年高考中，数列通常以解答大题形式出现，考查递推求通项、求和、放缩证明等。2026年新高考I卷将数列从解答题中移除，仅以一道填空压轴题的形式出现。这一变化释放出强烈信号：数列考查不再看重繁琐的代数运算和模板化的证明流程，而是转向**考查分析问题的思维方法和自主探究能力**。[1]

### 2. "重思轻算"的典型代表

本题几乎不涉及复杂计算，整道题的核心在于：
- 能否从给定的条件中**自主发现规律**
- 能否在多种可能性中**分类讨论、逐一排查**
- 能否将隐性的结构特征**显性化**

全程依靠层层递进的自问自答推进解题，而非列式演算。这与传统数列题"先算后验"的模式截然不同，体现了命题人对思维品质的重视。[2]

### 3. 隐蔽考查等差与等比的综合理解

题目并未直接说明原数列是等差数列或等比数列，但通过 $a_{3n} = n^2 + n$ 这一条件，隐蔽地给出了数列的**分组结构**。学生需要自主发现：将原数列按每3项一组划分，各组之和构成等差数列。这种隐蔽性有效区分了"会背公式"和"会分析问题"的学生。

### 4. 强区分度

本题作为填空压轴，具有竞赛风格，是区分中档生与优等生的关键一题。没有固定模板可套，审题不细心、基础不牢的考生极易踩坑丢分。

## 三、考查知识点梳理

| 知识模块 | 具体内容 | 考查深度 |
|---------|---------|---------|
| **数列通项** | 由条件推导数列项之间的关系 | 高 |
| **等差数列** | 等差数列的识别与通项公式 | 中 |
| **等比数列** | 等比数列的定义、公比的概念 | 高 |
| **分组构造** | 将数列按规则分组，构造新数列 | 极高 |
| **分类讨论** | 按首项在不同组中的位置分类 | 高 |
| **排除法** | 利用矛盾排除不可能的情形 | 高 |
| **最值分析** | 在可行解中求最大值 | 中 |

### 核心知识点详解

**1. 错位相减发现等差结构**

由 $a_{3n} = n^2 + n$ 和 $a_{3(n+1)} = (n+1)^2 + (n+1)$，作差可得：

$$a_{3n+3} - a_{3n} = (n+1)^2 + (n+1) - n^2 - n = 2n + 2$$

进一步由 $a_{3n} - a_{3(n-1)} = 2n$，可得：

$$a_{3n} + a_{3n-1} + a_{3n-2} = n^2 + n$$
$$a_{3(n-1)} + a_{3(n-1)-1} + a_{3(n-1)-2} = (n-1)^2 + (n-1) = n^2 - n$$

因此：

$$a_{3n} + a_{3n-1} + a_{3n-2} - (a_{3n-3} + a_{3n-4} + a_{3n-5}) = (n^2 + n) - (n^2 - n) = 2n$$

这就是关键结论：**原数列每3项之和构成等差数列**，首项和为 $2$，公差为 $2$。

**2. 连续9项与分组的关系**

9项连续项跨越3个完整的3项组。设这9项从某位置开始，根据起始项在第 $k$ 组中的位置，可分为不同情形。

**3. 等比数列的约束条件**

若连续9项 $\{c_1, c_2, \ldots, c_9\}$ 构成等比数列，则满足 $c_{i+1}/c_i = q$（$i = 1, 2, \ldots, 8$），共8个比值相等。同时，每连续3项的和（即对应一个组的和）构成等差数列。

## 四、解题方法与技巧

### 核心解题框架：分组模型法

**第一步：构建分组数列**

将原数列按每3项一组划分：

$$\text{第 } n \text{ 组}：\{a_{3n-2}, \; a_{3n-1}, \; a_{3n}\}$$

设第 $n$ 组的三项之和为 $b_n$：

$$b_n = a_{3n-2} + a_{3n-1} + a_{3n} = n^2 + n$$

**第二步：发现 $b_n$ 的等差性质**

$$b_{n+1} - b_n = [(n+1)^2 + (n+1)] - [n^2 + n] = 2n + 2$$

等等——这并非简单的等差数列！$b_n = n^2 + n$ 本身是二次的。

但更关键的是：每组的**完整三项之和**满足递推关系。

实际上，由条件可知每个"三项块"的和为 $2n$（$n$ 为组号），且相邻组的和的差为常数 $2$。也就是说，如果我们把连续9项分成三组，每组的和依次为 $S_1, S_2, S_3$，则：

$$S_2 - S_1 = d, \quad S_3 - S_2 = d$$

即三组和构成等差数列。

**第三步：等比数列的约束**

设等比数列的公比为 $q$，连续9项为：

$$t_1, \; t_1 q, \; t_1 q^2, \; t_1 q^3, \; t_1 q^4, \; t_1 q^5, \; t_1 q^6, \; t_1 q^7, \; t_1 q^8$$

每3项一组，三组和分别为：

$$S_1 = t_1(1 + q + q^2), \quad S_2 = t_1 q^3(1 + q + q^2), \quad S_3 = t_1 q^6(1 + q + q^2)$$

设 $C = 1 + q + q^2$，则 $S_1 = t_1 C$，$S_2 = t_1 q^3 C$，$S_3 = t_1 q^6 C$。

由于 $C \neq 0$（等比数列公比 $q > 0$），三组和构成公比为 $q^3$ 的等比数列。但同时，三组和又必须满足等差约束。

**第四步：等差与等比的双重约束**

设三组和分别为 $S_1, S_2, S_3$，它们既构成等差数列又构成等比数列，这意味着它们必须是**常数列**或**仅含两项等差**。

更准确地说，由组的和为 $2m, 2(m+1), 2(m+2)$（$m$ 为起始组号），三个正整数的和构成等差数列。同时三个和又构成等比数列，即：

$$2(m+1)^2 = 2m \cdot 2(m+2)$$

化简得：

$$(m+1)^2 = 4m(m+2)$$

$$m^2 + 2m + 1 = 4m^2 + 8m$$

$$3m^2 + 6m - 1 = 0$$

解得 $m = \dfrac{-6 \pm \sqrt{36 + 12}}{6} = \dfrac{-6 \pm \sqrt{48}}{6} = \dfrac{-3 \pm 2\sqrt{3}}{3}$

这不是正整数，说明**完整三组和构成等比数列不成立**。

**第五步：重新理解分组方式**

关键洞察：连续9项的**起始位置**不必恰好对齐组的边界。9项可能跨越组的边界，其中前 $r$ 项属于第 $m$ 组，中间3项属于第 $m+1$ 组，后 $9-r-3$ 项属于第 $m+2$ 组。

需要按起始项在组内的位置进行分类讨论。

**第六步：分类讨论求最大公比**

根据起始位置的不同，分情况讨论等比数列的约束，利用各组项之和的等差关系和等比关系联立求解。在所有可行情形中，求出 $q$ 的最大值。

经过完整的分类讨论与排除，**最大公比 $q = \sqrt{2}$**，出现在特定的分组排布情形中。

### 技巧要点总结

| 技巧 | 具体说明 |
|-----|---------|
| **分组构造** | 将原数列按3项一组划分，发现组内和的规律 |
| **错位相减** | 通过相邻组条件差发现递推关系 |
| **排除法** | 证明从第1项开始不可能成立，缩小搜索范围 |
| **分类讨论** | 按起始项在组中的位置（余数）分类 |
| **等差等比互化** | 利用"既是等差又是等比"的特殊约束条件 |
| **整体思维** | 不逐项计算，而是从组和的整体约束出发 |

## 五、题目难度与失分分析

### 难度定位

| 维度 | 评价 |
|-----|------|
| 知识门槛 | 中——等差、等比数列为课内核心知识 |
| 计算门槛 | 低——全程计算量不大 |
| 思维门槛 | 极高——需要自主发现结构、层层递进推理 |
| 综合难度 | ★★★★（填空压轴，竞赛风格） |

### 主要失分原因

1. **误判为等比数列**：题目未说原数列是等比数列，习惯性套公式的学生直接掉坑[3]
2. **未能发现分组结构**：看不出 $a_{3n} = n^2 + n$ 隐含的3项分组规律
3. **分类讨论不完整**：遗漏起始项在不同位置的情形，导致漏解
4. **不会排除不可能情形**：无法证明某些起始位置不成立，盲目搜索
5. **计算失误**：在公比的求解过程中代数变形出错
6. **时间不够**：作为填空压轴，留给本题的时间有限，思维链条太长来不及完成

## 六、备考建议

### 1. 强化"审疑思算"的解题理念

本题是"审疑思算"十六字解题法的典型范例：[2]

- **审题生疑**：读题时主动抛出问题——"原数列是什么类型？""连续9项如何与已知条件关联？"
- **疑中再审**：带着问题回看条件，发现 $a_{3n}$ 的下标暗示了分组结构
- **思后存疑**：推导过程中产生新问题——"起始位置不同会怎样？"
- **思推动算**：每一步推理都为下一步计算奠定基础

### 2. 掌握三大核心思想

| 思想方法 | 在本题中的应用 | 训练建议 |
|---------|--------------|---------|
| **转化与化归** | 将隐性的分组结构显性化 | 练习从条件中提取隐藏结构 |
| **基本量思想** | 将多个变量统一为组号和组内位置 | 面对多变量时先找基本量 |
| **分类讨论** | 按起始位置余数分类，不重不漏 | 练习确定分类标准的训练 |

### 3. 数列专项突破方向

- **分组构造**：将数列按固定长度分组，构造新数列的性质
- **递推关系**：从给定的项之间关系推导出递推公式
- **排除法**：证明某些情况不成立，缩小搜索范围
- **特殊化→一般化**：先看具体例子找规律，再推广到一般情况

### 4. 适应数列考查方式的变革

2026年高考I卷将数列从解答大题移至填空题，这一趋势可能持续。学生需要：
- 不再依赖解答题的固定模板（递推→通项→求和→放缩）
- 适应"短题干、深思维"的填空压轴风格
- 注重数列的结构分析能力，而非代数运算能力

### 5. 考场策略

- 填空压轴题预留3-5分钟
- 先判断起始位置的可行性（如第1项开始是否可能），缩小讨论范围
- 若时间不足，可尝试特殊值法猜测答案，但要注意验证
- 保持冷静，不因题型陌生而放弃

## 七、总结评价

2026年高考数学I卷第14题是一道**极具思维深度的高质量填空压轴题**，其特点可概括为：

1. **重思轻算**：几乎不涉及复杂计算，全程依靠思维推进，是"多想少算"理念的极致体现
2. **隐蔽性强**：等差等比的约束以间接方式给出，需要自主发现规律
3. **竞赛风格**：分析问题的一般方法（观察→归纳→验证→推广）与科研方法一致
4. **反套路**：未直接标注等比数列，打破"看题干就套公式"的习惯

本题折射出的核心解题方法论远超数列本身：**先观察数据，再总结规律，最后理论推导。** 这种从具体到抽象、从特殊到一般的分析路径，不仅是数学学习的基本方法，更是未来科研和实际工作中解决问题的通用能力。

本题对新高考命题方向的启示是明确的：**未来的高考不再考查"你背了多少公式"，而是考查"你能不能用数学的方式思考问题"。**
