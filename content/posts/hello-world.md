## 欢迎来到 Hugo！

这是使用 [Hugo](https://gohugo.io/) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod) 主题搭建的静态博客。

### Hugo 的优势

- **极速构建**：毫秒级生成静态页面
- **部署简单**：配合 Cloudflare Pages 实现自动部署
- **Markdown 写作**：专注于内容创作

```bash
# 新建文章
hugo new content posts/my-post.md

# 本地预览
hugo server

# 构建发布
hugo
```

享受写作的乐趣吧！ ✍️
