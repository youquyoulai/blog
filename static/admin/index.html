使用说明

1. 图库：上传图片自动到 R2，点击图片可复制链接
2. 图片上传：自动生成 orig / 800 / 300 三种尺寸的 WebP 版本，全部免费无额外费用；插入时用 -800.webp 是文章配图，-300.webp 是缩略图；可取消勾选"自动压缩"上传原图
3. 文章：用 @image:图片名.png 语法插入图片
4. 发布：点击"发布"按钮触发博客重新构建
5. 评论：前往 Waline 后台 管理评论
